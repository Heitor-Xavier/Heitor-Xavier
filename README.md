## 📂 Projetos em Destaque

### 📦 [Análise de Inventário Logístico](https://github.com/Heitor-Xavier/analise-inventario-)
> SQL + Python + Excel

Análise completa de estoque com queries SQL para classificação por status, giro, cobertura e Curva ABC. Gera relatório Excel formatado automaticamente.

`Python` `Pandas` `SQL` `SQLite` `OpenPyXL`

---

### 🔄 [Reconciliação de Estoque — DHL x Cliente](https://github.com/Heitor-Xavier/reconciliacao-estoque)
> Comparação de bases Excel com geração de relatório e envio por e-mail

Compara duas bases de estoque (sistema DHL vs sistema do cliente), identifica divergências de saldo, entradas e saídas, valida snapshots e gera Excel formatado enviado por e-mail automaticamente.

`Python` `Pandas` `OpenPyXL` `smtplib`

---

### 📊 [Automação de Relatório Logístico](https://github.com/Heitor-Xavier/automacao-relatorio)
> Pipeline completo Excel → PDF → E-mail

Pipeline que lê dados de um Excel, calcula KPIs logísticos (OTD, NPS, Lead Time, Custo de Frete), gera gráficos e monta um PDF formatado enviado automaticamente por e-mail.

`Python` `Pandas` `Matplotlib` `FPDF2` `OpenPyXL` `smtplib`
